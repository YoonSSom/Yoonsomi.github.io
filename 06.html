고객 리뷰 기반 상품 추천 시스템 개발
"리뷰 한 줄이 다음 시즌을 만든다"

🛠 사용 기술

언어: Python
라이브러리: Pandas, Huggingface Transformers
모델: KoBERT, KoGPT2
기법: Easy Data Augmentation (RI, RR, RD), Back Translation, Keyword-based Generation

📌 프로젝트 개요

고객 리뷰 데이터를 기반으로, 무신사(MUSINSA) 입점 브랜드의 신뢰도 높은 평가지표를 만들고
다음 분기에 추천할 상품을 예측하는 프로젝트를 수행했습니다.
리뷰 감성 분석, 별점 예측, 추천 알고리즘을 결합하여 실질적인 인사이트를 제공하는 데 주력했습니다.

🧩 주요 역할

- 데이터 수집 및 전처리
- 별점 편중 문제 해결
- 리뷰 데이터의 편향(1~3점 부족)을 해소하기 위해, **EDA(Easy Data Augmentation)**의 RI, RR, RD 방식을 사용해 각 점수당 4만 개씩 데이터를 증식
또한 Back Translation 기법(Googletrans 활용)으로 원문을 외국어로 번역 후 다시 한글로 되돌려 데이터 다양성 확보

🧠 모델링 및 감성 분석

- KoBERT 기반 감성 분석 모델을 사용해 긍정/부정 리뷰 자동 분류 및 별점 재정의
- 리뷰 문장의 어휘 유사도 및 흐름을 반영한 KoGPT2 기반 Generation 기법을 도입, 문장 내 키워드 앞뒤 문맥을 고려해 데이터 생성

🔍 예측 및 추천

- KoBERT로 분석한 리뷰 감정과 예측된 별점을 활용해, 입점 브랜드별 평가지표 생성
- 이를 기반으로 다음 분기 추천 아이템 리스트 도출

✅ 주요 성과

- 감성 분석 + 별점 예측 기반의 ‘신뢰성 높은 평가지표’ 제공
- 데이터 예측 모델을 통해 다음 분기 추천 상품 리스트 정확도 향상
- 리뷰 기반 추천 시스템의 실질적 적용 가능성 검증

💡 프로젝트를 통해 얻은 인사이트

단순한 리뷰 분석을 넘어, 텍스트 증식 → 감성 분석 → 평점 예측 → 추천 시스템이라는 데이터 활용의 전 과정을 경험했습니다.
특히 데이터가 부족한 상황에서도 증식 기법을 적극 활용하며 자연어 처리(NLP)의 실무적 응용 능력을 크게 향상시킬 수 있었습니다.
