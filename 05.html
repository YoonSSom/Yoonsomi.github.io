"픽셀 하나 놓치지 않는 시선"
Semantic Segmentation 기반 파손 인식

🛠 사용 기술

언어: Python
라이브러리: PyTorch, Albumentations, NumPy, Pillow, Matplotlib
모델: DeepLabV3
툴 & 플랫폼: VIA Tool, Weights & Biases, AWS EC2/S3

📌 프로젝트 개요

자동차 외관 이미지로부터 파손 부위를 자동으로 탐지하는 Semantic Segmentation 프로젝트입니다.
DeepLabV3 모델과 자체 구축한 이미지 데이터셋을 활용해, 파손 영역(스크래치/찌그러짐 등)을 픽셀 단위로 분류하고 시각화했습니다.

🧩 주요 역할

- 웹에서 자동차 외관 이미지 수집 및 직접 촬영
- VIA 툴을 이용해 Segmentation용 데이터 라벨링 수행
- Albumentations를 사용해 다양한 오프라인 Augmentation 적용
- Binary & Multi-class segmentation 실험 병행
- DeepLabV3 모델 튜닝 및 fine-tuning 수행
- 학습 로그를 W&B로 관리하고 시각화

🧠 모델링 및 기술적 접근

- DeepLabV3 Pretrained 모델 기반으로 Reference 코드 수정
- 하이퍼파라미터 튜닝 및 best checkpoint 저장
- 불균형한 클래스 대응을 위해 class weight 보정 및 data balancing 적용
- 학습 후 예측 이미지를 시각화하여 실제 파손 탐지 성능을 정성적으로 평가

🔍 실험 결과 및 분석

- 실제 이미지에서 파손 부위를 정확히 분할하여 시각적 출력 가능
- 소량의 데이터셋을 증식하여도 Semantic segmentation 성능을 일정 수준 유지
- Augmentation 기법의 중요성과 효과를 체감하며 활용 경험 확대

✅ 주요 성과

- 실제 자동차 파손 부위 별 탐지 성공 및 시각화
- Segmentation 모델 적용 및 튜닝 경험 확보
- 학습-추론-시각화 전체 파이프라인 경험

💡 프로젝트를 통해 얻은 인사이트

직접 데이터셋을 구축하고 세분화된 클래스까지 고려한 Segmentation을 진행하며 CV 프로젝트의 A to Z를 경험했습니다.
특히 부족한 데이터 환경에서도 증식을 통한 성능 향상 가능성을 체감했고, 추후 실서비스 연계 가능성까지 고려한 프로젝트로 발전시켰습니다.
