"맛있는 번역기"
메뉴판을 읽는 OCR 서비스

🛠 사용 기술

언어: Python
라이브러리: PyTorch, Albumentations, NumPy, Pillow, Matplotlib
모델: YOLO (Object Detection), OCR (Text Recognition), NLP (Papago API 활용)
툴 & 플랫폼: Computer Vision (CV), OpenCV, PyTorch

📌 프로젝트 개요

자포스트 코로나 시대 외국인 관광객을 위한 메뉴판 자동 번역 서비스 개발
메뉴판 이미지에서 텍스트 탐지 → 인식 → 번역까지 End-to-End 파이프라인 구현

🧩 주요 역할

- 팀 프로젝트(5인)에서 Detection 전담 & Recognition 보조
- YOLO 모델 적용 및 성능 최적화, 데이터 수집·전처리 및 모델 Follow-up 수행

🧠 모델링 및 기술적 접근

- YOLO 기반 메뉴판 텍스트 Detection 모델 선정 및 Custom Training
- 세로형 메뉴판 Detection 성공, 기울어진 메뉴판 탐지 성능 개선 진행

🔍 실험 결과 및 분석

- Super Resolution을 통해 저화질 이미지 OCR 성능 개선
- YOLO 기반 Detection으로 안정적인 텍스트 영역 추출
- Papago API와 연동해 실사용 가능한 번역 서비스 프로토타입 구현

✅ 주요 성과

- Detection 성능 향상 주도 → OCR 전체 파이프라인 성능 개선에 기여
- End-to-End 번역 서비스 구현 → 외국인 관광객 활용 가능성 입증
- OCR 기술의 다양한 문서(명함, 영수증, 문서 등) 확장 가능성 검증

💡 프로젝트를 통해 얻은 인사이트

한정된 리소스 환경에서도 적절한 모델 선정·전처리 전략의 중요성 체득
Detection, OCR, NLP를 연계한 서비스형 파이프라인 구축 경험 확보, 기술 연구를 넘어 실사용 서비스 구현 역량 강화
